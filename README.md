# 🏋️‍♀️ GymLogger

A modern, mobile-first fitness tracking application built with **Next.js** and **Firebase**. This app allows users to manage workout routines, track exercises, and save progress seamlessly.

## 🚀 Key Features
- **Routine Management:** Pre-loaded with "Majestic" full-body routines (A, B, and C).
- **Hybrid Storage:** Supports **Demo Mode** using browser LocalStorage and **Cloud Mode** via Firebase.
- **Mobile First:** Designed to be installed as a PWA (Progressive Web App) on smartphones.
- **Real-time Updates:** Instant UI updates using React Context and Hooks.

## 🛠 Tech Stack
- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Database & Auth:** [Google Firebase](https://firebase.google.com/)
- **Deployment:** [Vercel](https://vercel.com/)

## 📱 How to Use on Your Phone
1. Open the [Live Demo Link](REPLACE_WITH_YOUR_VERCEL_URL) in your mobile browser.
2. Tap the **Share** button (iOS) or **Menu** (Android).
3. Select **"Add to Home Screen"**.
4. Launch GymLogger from your home screen like a native app!
