# 🏋️‍♂️ Gym Logger - Workout Tracker

A modern, full-stack web application designed to help users track their strength training progress with ease. This project focuses on simplicity, data persistence, and a seamless user experience.

## ✨ Key Features
- **Secure Authentication:** Integrated Google Sign-In using **Firebase Auth**.
- **Real-time Logging:** Track movements, weights, and reps instantly.
- **Dynamic Dashboard:** View total sets and volume (total weight lifted) per session.
- **Persistent Storage:** Data is securely stored and retrieved using **Firebase Firestore**.
- **Responsive Design:** Fully optimized for both mobile and desktop use using **Tailwind CSS**.
- **History Tracking:** Access past workouts to monitor progress over time.

---

## 🛠 Tech Stack
- **Frontend:** React.js / Next.js
- **Styling:** Tailwind CSS
- **Backend/Database:** Firebase Firestore
- **Authentication:** Firebase Auth
- **Deployment:** Vercel
